Destination Path

This document describes the role of the destination path in processing backlog transactions in Karaf incidents.

The destination path is the full path to the processing folder. This is usually where the blockage happened during the incident. All files in this folder will be processed by the Apache Karaf agent.

For example: C:\TEST\folder2
