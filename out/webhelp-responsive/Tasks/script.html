Processing files with .bat script

This document explains how to use a .bat script to process backlog transactions Karaf incidents.

Ensure that the files to be processed are stored in the (source path).

To automatically process transactions in the backlog using the .bat script, follow the steps below:

  1. On the Karaf server, navigate to the path C:\KARAF_TROUBLESHOOTING.
  2. Double-click on move_karaf_files_script.bat script file, provide the source path and the destination path when prompted and press enter.
The transactions will automatically move from source path to destination path one at a time, with a five second delay between each transaction.
